📄 README - Table Layout Webpage
📌 Overview

This project is a simple HTML webpage that demonstrates how to create a basic website layout using an HTML table. It includes different sections such as a logo area, content area, menu, and footer.

🛠 Features
Structured layout using <table>
Sections included:
Website Logo
Content Area
Menu
Footer
Uses table attributes:
border
cellspacing
cellpadding
Inline CSS styling for:
Border color
Text alignment
Font styling
Width control
Center-aligned layout
💻 Technologies Used
HTML5
Inline CSS
📂 File Structure
index.html – Main file containing the table layout code
🚀 How to Run
Save the file as index.html
Open it in any web browser
View the structured table-based webpage layout
🎯 Purpose

This assignment helps in understanding:

How to design webpage layouts using HTML tables
Basic structure of a webpage (header, content, menu, footer)
Use of table attributes and inline styling
Beginner-level webpage structuring concepts
📚 Learning Outcome

After completing this project, you will understand how HTML tables can be used to create structured webpage layouts and how basic styling improves presentation.
